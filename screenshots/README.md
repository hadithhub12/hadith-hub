## Screenshot Guidelines

1. Use a clean phone/simulator with no personal data visible
2. Recommended resolution: 390x844 (iPhone 14) or similar
3. PNG format, optimized for web
4. Crop to show only relevant UI elements
5. Add subtle device frames if desired (optional)
